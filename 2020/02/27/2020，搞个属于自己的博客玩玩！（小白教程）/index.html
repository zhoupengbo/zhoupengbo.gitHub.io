2020，搞个属于自己的博客玩玩！（小白教程）

发表于 2020-02-27 分类于 小玩意儿 阅读次数： Valine：
本文字数： 3.7k 阅读时长 ≈ 3 分钟

前言：之前一直羡慕别人都有自己的博客，或者说个人主页，2020了，咱也搞一个玩玩！本教程面向每一个想拥有它的人，不需要有什么基础，不需要懂什么IT技术，当然也不需要去买一个服务器或花你一分钱。你只需要懂得下载、安装、注册、复制、粘贴，然后照着教程走，就一定能拥有一个属于自己的Blog Or Zone！

开篇之前，先让大家看看我在2020年刚刚为自己搭建的专属博客吧！不喜勿喷哦~
博客地址：https://zhoupengbo.gitee.io/，大家可以在浏览器地址栏访问一下看看效果。
支持功能：文章分类、标签云、文章归档、站内搜索、文章评论、打赏、阅读数统计、cdn、自动生成文章目录 等。

首页图展示效果如下所示：

大量的坑我已踩过，大家不必从零做起，简单说只需复制我的博客，然后做稍微改动就可以！好了，想动手的朋友下面就跟着我的脚步去搞一个全网所有用户都可以访问到的以你自己的名字或是你喜欢的名字命名的博客系统吧！为了大家能够更好更简单理解和操作，本文还为大家附上了大量贴图~

技术架构： Hexo + NexT + Gitee Pages （GitHub Pages国内速度太慢，也可以选择Coding Pages，不懂不熟没关系，接着看下文）

1. 软件包下载

构建博客所需软件包我已整理好存于百度云，永久有效，请在计算机中任意选择一个目录下载资料包（建议选择磁盘根目录）。下载链接如下：

下载完后是一个名为myblog的文件夹，如图：

此时，我的文件夹全路径为：Z:\myblog，目录下为所需软件资料，如图：

下面我们开始软件包的安装工作，就像安装微信一样简单。

2. 软件包安装

注意：此后所有操作全部在根目录Z:\myblog下进行。

2.1 Git 安装

安装文件：Z:\myblog\Git-2.24.0-64-bit.exe
双击进行安装，一路next即可。（可自定义安装路径）
如有不明，请参考网上这篇详细文档：
https://blog.csdn.net/qq_32786873/article/details/80570783
验证安装是否成功：选择任意目录，右击，选择Git Bash Here选项，如图：

出现如下窗口，输入git –version 返回Git 版本即安装成功，如图：

2.2 Node.js 安装

安装文件：Z:\myblog\node-v12.14.1-x64.msi
双击进行安装，一路next即可。（可自定义安装路径）
如有不明，请参考网上这篇详细文档：
https://www.cnblogs.com/zhouyu2017/p/6485265.html
验证安装是否成功：选择任意目录，右击，选择Git Bash Here选项，输入node -v 返回版本即安装成功，如图：

2.3 HexoEditor 编辑器安装

安装文件：Z:\myblog\HexoEditor.zip
解压文件到当前文件夹，进入HexoEditor目录，找到start.cmd文件，依次点击【右击】【发送到】【桌面快捷方式】，这时候桌面已经生成快捷方式，我们双击打开（稍等片刻），点击左下角图标打开后如下图：

这就是我们以后写博客的编辑器了，后面会详细介绍如何使用。

3. Pages 服务启动

3.1 码云注册

先登陆码云注册平台https://gitee.com/signup 注册一个账号，这里你需要一个属于自己的邮箱账号进行注册。
填写姓名，默认个人空间名默认是姓名，请保持一致，填写邮箱，验证码，密码完成注册。如图：

3.2 绑定手机号

完成注册我们进入码云主页面，点击右上角头像，依次选择【设置】【绑定号码】完成手机号绑定。如图：

显示如下说明绑定成功！

3.3 新建仓库

不懂没关系，按图操作即可。可以简单理解为新建一个项目。
点击右上角 + 号，选择【新建仓库】，按下图进行操作：

点击创建即可，至此仓库创建完毕。

3.4 开启Pages服务

我们的博客要想正常运行，需要托管在此服务上。创建仓库完毕后页面会自动跳转至仓库页面，如下图。我们选择【服务】【Gitee Pages】【启动】，启动完毕后记得选择免费试用一个月（免部署），并记好自己的博客地址哦！

一个月到期后，如果我们更新博文，需要到这里重新部署服务才会更新哦~

下面我们访问一下自己的博客地址看看丑不丑吧！哈哈哈~

这就是我们的初始页面，相当丑，下面我们就来美化一下它，将我自己的博客模板套上去。

4.1 设置头像

Z:\myblog\blog\themes\next\source\images\avatar.gif

4.2 修改博客配置

打开博客配置文件：Z:\myblog\blog_config.yml
1） 更改为自己的博客名称、描述、作者。

2） 将所有的 ‘yoursite’ 替换为你的仓库名，例如我的仓库名为：‘luckyzpb’。
3） 保存文件，关闭文件。

最后说一下HexoEditor工具的使用。打开HexoEditor，点击左下角，点击新建Post，然后就可以写文章了。如图：
